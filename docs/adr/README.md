# Architecture Decision Records

This directory holds Architecture Decision Records (ADRs): short documents that
capture an architecturally significant decision, the context that led to it, and
its consequences.

Each record is a numbered Markdown file (`NNNN-title.md`). They are immutable once
accepted — to change a decision, add a new ADR that supersedes the old one and
update the old one's status.
